Canvasrama

Olá, bem vindo ao Canvasrama! Um fliperama online de jogos clássicos refeitos com HTML, CSS e Javascript, usando o Canvas. Feito como um lugar pra guardar meus estudos do livro A Lógica do jogo de Marcus Becker. É claro, no livro ele ensina a fazer os jogos usando Java, mas decidi adaptar para Canvas pra ser mais divertido.
